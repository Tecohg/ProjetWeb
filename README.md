HTML
CSS 
JAVASCRIPT
PHP

Deux idées:
Un escape game a deux joueurs, les joueurs sont dans deux salles differentes, et doivent communiquer entres eux via un chat, les indices pour une salle sont dans l'autre, progresser dans une salle (en faisant des puzzles) donne accès à d'autres parties du jeu(indices, histoire, puzzles) dans l'autre salle.


exemple d'un debut de partie basique:
j1: devant lui un keypad avec 9 symboles et 4 cases
j2: devant lui un script contenant une histoire
j1: explique a j2 via un chat ce qu'il voit devant lui
j2: en comprenant ce que j1 à devant lui, il lit le script et voit que quelques symboles correspondent a des evenements de l'histoire
j2: explique quels symboles pourrait etre mis chronologiquement
j1: trouve la bonne combinaison, le puzzle s'ouvre avec une nouvelle page
j2: une boite se debloque contenant une carte et des pions, ainsi qu'une note avec des mots... 


Salle d'attente pour deux joueur 
liste de serveurs (creer ou rejoindre rajouter la possibilité d'accepter ou non)
identifiant de partie(soit par sauvegarde (BDD) ou sinon temporaire en session)
